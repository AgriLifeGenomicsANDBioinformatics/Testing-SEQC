Description:
  Computes fragment length distribution and plots it.

Usage:
  get_fragment_length_distribution.sh  [options]  -- BEDGRAPH_FILE/TXT_FILE

Options:
  -h|--help           Help
  -d|--outdir         Output directory
         
Examples:
  get_fragment_length_distribution.sh -d fragment_length_distribution -- r1s1.bedGraph.gz 
  get_fragment_length_distribution.sh r1s1.txt

Dependancies:
  bedtools
  ggplot2
